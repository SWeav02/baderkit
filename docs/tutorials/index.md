# Tutorial Introduction

This module provides tutorials showing the complete process from start to finish using VASP or quantum espresso then moving to BaderKit. The files used in each tutorial can be downloaded [here](https://github.com/SWeav02/baderkit/releases/tag/0.9.0 ). Each folder contains the input and output files required for the tutorial as well as the full baderkit python script used in the tutorial.

!!! Note
    We do not provide `POTCAR` files as VASP does not permit their distribution. If you use VASP you will have to provide these yourself.

The files can also be downloaded through the command line. Note that this will download all examples which are ~0.75 GB and ~3.25 GB when unzipped.

```bash
baderkit download-examples
```
