# Tutorial Introduction

We provide examples of using each method. The files used in each tutorial can be downloaded [here](https://github.com/SWeav02/baderkit/releases/tag/0.9.0 ). Each folder contains the VASP or .cube files required for the tutorial as well as the full python script used. We only include examples for the python API as the command-line usage follows a simple pattern described in our [Basic Usage](/usage) page.

The files can also be downloaded through the command line. Note that this will download all examples which are ~0.75 GB and ~3.25 GB when unzipped.

```bash
baderkit download-examples
```
